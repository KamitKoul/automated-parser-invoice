# Automated Invoice Parser

This project is an automated invoice parser with a React frontend and a Node.js/Express backend. It uses OCR and NLP to extract information from uploaded invoices.

## Project Structure

- `backend/`: Node.js server, controllers, models, and routes.
- `frontend/`: React application with Vite, Tailwind CSS, and Material UI.

## Getting Started

### Backend

1. Navigate to the `backend` directory.
2. Install dependencies: `npm install`
3. Start the server: `npm start`

### Frontend

1. Navigate to the `frontend` directory.
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`

## Features

- User authentication (Login/Register)
- Invoice upload and processing
- Dashboard for viewing processed documents
- Document details view with extracted data
